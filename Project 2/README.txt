Unsupervised Learning

Make sure to have an "Images" folder to store the images.

To run the adult data set:
python adult.py

To run the water treatment data set for cluster visualizations:
python water_treatment.py

To run the water treatment data set for charts:
python watertreatment.py

You need to have pandas, numpy, scipy, tensorflow, and scikit-learn installed for Python 3.5+.

You can comment out any of the method calls in the main function to selectively call certain cluster and dimensionality reduciton algorithms.
