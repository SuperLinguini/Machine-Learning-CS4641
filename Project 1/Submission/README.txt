To run the adult data set:
python adult.py

To run the diabetes data set:
python diabetes.py

You need to have pandas, numpy, scipy, tensorflow, and scikit-learn installed for Python 3.5+.

You can comment out any of the method calls in the main function to selectively call certain algorithms.

Diabetes 130 in US Hospitals from 1999-2008 Data Set
https://archive.ics.uci.edu/ml/datasets/Diabetes+130-US+hospitals+for+years+1999-2008

Adult Census Income Data Set
https://archive.ics.uci.edu/ml/datasets/Adult
